# 使用指南

## 基础使用

### 场景 1: 网页划词翻译

1. 在网页中选中英文文本。
2. 等待悬浮卡片返回译文，或按 `Alt+T` 主动触发。
3. 在卡片中查看原文、译文与术语命中结果。

### 场景 2: YouTube 字幕双语翻译

1. 打开 YouTube 视频并启用字幕。
2. 插件会在播放器区域渲染双语字幕层。
3. 连续重复字幕会优先命中缓存，减少重复请求。

### 场景 3: 配置模型 API

1. 打开扩展 `Options` 页面。
2. 填写 `Base URL`、`API Key`、`Model`。
3. 保存并执行连接测试。

## 高级用法

### 术语表管理

1. 在 `Term Editor` 新增术语映射（Source / Target）。
2. 保存后将参与翻译后处理，提升专有名词一致性。

### 缓存统计与清理

1. 在统计区域查看缓存命中数据。
2. 调试或回归测试时可清空缓存。

## 常见问题

### Q: 为什么整页翻译和 PDF 翻译没有效果？
A: 当前版本仅交付划词与 YouTube 场景，`src/content/page.ts` 和 `src/content/pdf.ts` 仍是占位实现。

### Q: API 测试失败如何排查？
A: 依次检查 `Base URL`（HTTPS）、`API Key` 有效性、模型名称是否存在。

### Q: YouTube 不显示双语字幕层怎么办？
A: 先确认视频字幕已开启，再刷新页面重试；部分视频需等待字幕轨加载后触发翻译。

## 故障排查

### 问题: 返回 401 认证失败

**症状**: 无译文或提示认证失败。  
**原因**: API Key 无效、过期或与 Base URL 不匹配。  
**解决**: 在 Options 页更新 API Key，重新执行连接测试。

### 问题: 字幕翻译延迟偏高

**症状**: 字幕译文明显滞后。  
**原因**: 网络抖动、模型响应慢、缓存未命中。  
**解决**: 切换更快模型、使用稳定网络、观察缓存命中率后再评估。
